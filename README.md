# Website_Sdg_13_Calvin25
Awareness campaign website for sdg 13
Climate Action Hackathon 2025 Website

This project is a responsive event webpage for the Climate Action Hackathon 2025 – Urban Heat & Flood Resilience.
It provides information about the event, speakers, schedule, resources, and includes a registration form.

Features

Responsive navigation bar

Hero section with event details

About section

SDG 13 explanation

Event schedule

Speakers & mentors

Challenge tracks

Resources

Registration form

Sponsors & partners section

Footer with contact info

Files Included

index.html – Main structure of the webpage

style.css – Styling and layout of the webpage

images/ – Contains images used in the design (Event, Speakers, Sponsors)

How to Use

Download all files and keep them in the same folder.

Make sure the images folder is correctly placed.

Open index.html in your browser.

Technologies Used

HTML

CSS

Flexbox layout

Responsive design principles

Customization

You can easily update:

Event title and details

Speaker images and info

Schedule dates

Links and resources

Colors and fonts in style.css
